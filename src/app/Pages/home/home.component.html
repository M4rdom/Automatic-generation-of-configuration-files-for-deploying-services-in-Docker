SPL: Automatic generation of configuration files

Generate Custom Files Using UVLS extension and Templates

This Website is used to generate product examples using Jinja Templates, UVengine and UVLS extension. Currently it relies on GitHub Public Repository to store the templates and the generated products. The repository is public and can be accessed by anyone.

Create Configuration

Select an existing template or push a new one to the GitHub repository.

Create a new configuration using the UVLS extension by downloading a feature model file. See UVLS.

Generate

Paste the configuration JSON and wait until the product is generated.

Feel free to modify and add the final touches to the product to suit your needs. (It might be mandatory depending on the template.)

Download Product

Download the final product and customize the name and extension if desired

or just copy the content on the product and go
